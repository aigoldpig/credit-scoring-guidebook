    к°ңмҡ”мҷҖ Pre-binningВ¶

    1.1 лқјмқҙлёҢлҹ¬лҰ¬ к°ңмҡ”мҷҖ м•„нӮӨн…ҚмІҳВ¶

    optbinningмқҖ мҲҳн•ҷм Ғ мөңм Ғнҷ”(нҳјн•©м •мҲҳкі„нҡҚлІ• MIP / м ңм•Ҫ н”„лЎңк·ёлһҳл°Қ CP)лЎң binning л¬ём ңлҘј н’Җм–ҙ, лӢЁмЎ°м„ұВ·мөңмҶҢ мғҳн”ҢВ·нҶөкі„м Ғ мң мқҳм„ұ л“ұ м—¬лҹ¬ м ңм•Ҫмқ„ лҸҷмӢңм—җ л§ҢмЎұн•ҳлҠ” мөңм Ғ кө¬к°„мқ„ м°ҫлҠ”лӢӨ.

    мІҳлҰ¬ нҢҢмқҙн”„лқјмқёВ¶

    Pre-binning вҶ’ мөңм Ғнҷ”(MIP/CP) вҶ’ Binning Table вҶ’ WoE ліҖнҷҳ вҶ’ (м„ нғқ) мҠӨмҪ”м–ҙм№ҙл“ң

    лӢЁкі„ м—ӯн•  н•өмӢ¬ нҒҙлһҳмҠӨ
    Pre-binning мҙҲкё° Fine Bin мғқм„ұ (CART, quantile л“ұ) OptimalBinning лӮҙл¶Җ
    мөңм Ғнҷ” м ңм•ҪмЎ°кұҙ н•ҳм—җм„ң IV мөңлҢҖнҷ”н•ҳлҠ” н•©лі‘ мЎ°н•© нғҗмғү OptimalBinning
    Binning Table кө¬к°„лі„ нҶөкі„лҹү м¶ңл Ҙ (Count, WoE, IV л“ұ) binning_table
    WoE ліҖнҷҳ мӣҗліё лҚ°мқҙн„°лҘј WoE к°’мңјлЎң м№ҳнҷҳ transform()
    лӢӨліҖлҹү мІҳлҰ¬ м „мІҙ ліҖмҲҳ мқјкҙ„ binning + ліҖмҲҳ м„ лі„ BinningProcess
    мҠӨмҪ”м–ҙм№ҙл“ң лЎңм§ҖмҠӨнӢұ нҡҢк·Җ + м җмҲҳ ліҖнҷҳ Scorecard

    1.2 Pre-binning вҖ” мҙҲкё° кө¬к°„ мғқм„ұВ¶

    мөңм Ғнҷ” м „м—җ м—°мҶҚнҳ• ліҖмҲҳлҘј мҙҲкё° Fine BinмңјлЎң лӮҳлҲ„лҠ” лӢЁкі„лӢӨ. мқҙ лӢЁкі„мқҳ н’Ҳм§Ҳмқҙ мөңмў… кІ°кіјм—җ м§Ғм ‘ мҳҒн–Ҙмқ„ лҜём№ңлӢӨ.

    л°©мӢқ prebinning_method мӣҗлҰ¬ нҠ№м§•
    CART (кё°ліёк°’) "cart" DecisionTreeClassifierлЎң м •ліҙ мқҙл“қ кё°л°ҳ л¶„н•  нғҖкІҹ л°ҳмқ‘лҘ  м°Ёмқҙк°Җ нҒ° м§Җм җмқ„ мһҗлҸҷ нғҗм§Җ, к°ҖмһҘ м Ғмқ‘м Ғ
    MDLP "mdlp" мөңмҶҢ кё°мҲ  кёёмқҙ мӣҗлҰ¬ кё°л°ҳ м—”нҠёлЎңн”ј л¶„н•  мқҙлЎ м ҒмңјлЎң м •көҗн•ҳлӮҳ CART лҢҖл№„ мӢӨл¬ҙ м°Ёмқҙ лҜёлҜё
    Quantile "quantile" л“ұл№ҲлҸ„(equal-frequency) л¶„н•  к°Ғ кө¬к°„мқҳ мғҳн”Ң мҲҳ к· л“ұ, л№„м§ҖлҸ„ л°©мӢқ
    Uniform "uniform" л“ұк°„кІ©(equal-width) л¶„н•  к°’ лІ”мң„ кё°мӨҖ к· л“ұ л¶„н• , нҺён–Ҙлҗң л¶„нҸ¬м—җ м·Ём•Ҫ 
    optb = OptimalBinning(
    name="utilization_rate",
    prebinning_method="cart",   # кё°ліёк°’, лҢҖл¶Җл¶„ мқҙлҢҖлЎң мӮ¬мҡ©
    max_n_prebins=20,           # мҙҲкё° Fine Bin мөңлҢҖ мҲҳ (кё°ліё 20)
    min_prebin_size=0.05,       # Fine BinлӢ№ мөңмҶҢ мғҳн”Ң л№„мңЁ
)

    Pre-binning нӣ„ м •м ң

    Pre-binning кІ°кіјм—җм„ң мҲңмҲҳ Bin(Event=0 лҳҗлҠ” Non-event=0)мқҙ л°ңмғқн•ҳл©ҙ WoEк°Җ ВұвҲһк°Җ лҗңлӢӨ. optbinningмқҖ мқҙлҹ° мҲңмҲҳ Binмқ„ мқём ‘ Binкіј мһҗлҸҷ н•©лі‘н•ҳм—¬ м •м ңн•ң нӣ„ мөңм Ғнҷ” лӢЁкі„лЎң л„ҳкёҙлӢӨ.
