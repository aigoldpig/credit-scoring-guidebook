# 2. Functional ANOVA — 함수의 직교 분해

!!! warning "작성 중"
    이 페이지는 현재 작성 중입니다.

---

[1절](non-identifiability.md)에서 확인한 것처럼, 제약 없이는 하나의 함수를 무한히 많은 방식으로 분해할 수 있다. Functional ANOVA(fANOVA)는 **수학적 제약 조건을 부과하여 분해를 유일하게 결정**하는 프레임워크다.

## 분해의 정의

임의의 함수 \(\hat{f}(\mathbf{x})\)를 **모든 변수 부분집합의 효과 합**으로 분해한다:

$$
\hat{f}(\mathbf{x}) = f_0 + \sum_{i} f_i(x_i) + \sum_{i < j} f_{ij}(x_i, x_j) + \cdots + f_{1,\ldots,p}(x_1, \ldots, x_p)
\tag{1}
$$

각 component \(f_S\)는 **재귀적으로** 정의된다 (Hooker, 2004):

$$
f_S(\mathbf{x}_S) = \int_{X_{-S}} \left( \hat{f}(\mathbf{x}) - \sum_{V \subset S} f_V(\mathbf{x}_V) \right) d\mathbb{P}(X_{-S})
$$

이 수식이 실제로 무엇을 하는지, 각 component별로 풀어서 보자:

| Component | 수식 | 직관 |
|-----------|------|------|
| Intercept | \(f_0 = \mathbb{E}[\hat{f}(\mathbf{X})]\) | 전체 평균 |
| Main effect | \(f_1(x_1) = \mathbb{E}[\hat{f}(\mathbf{X}) \mid X_1 = x_1] - f_0\) | 나머지를 평균 낸 뒤, 전체 평균을 빼면 순수 효과 |
| 2차 interaction | \(f_{12} = \mathbb{E}[\hat{f} \mid X_1, X_2] - f_1 - f_2 - f_0\) | 하위 효과를 다 빼면 순수 교호작용 |

직관: **나머지 변수를 적분(marginalize)해서 없앤 뒤, 이미 설명된 하위 효과를 빼면 해당 집합만의 순수 효과가 남는다.**

### XOR 함수로 직접 계산해보기

[1절](non-identifiability.md)에서 네 가지 분해를 보았던 XOR 함수에 fANOVA를 적용하자. 균등분포를 가정한다 (4가지 입력이 동일 확률).

대상 함수: \(Y(0,0)=0,\; Y(0,1)=1,\; Y(1,0)=1,\; Y(1,1)=0\)

**Intercept — 전체 평균:**

$$
f_0 = \mathbb{E}[Y] = \frac{0 + 1 + 1 + 0}{4} = 0.5
$$

**Main effect — 나머지를 평균 내고, 전체 평균을 뺀다:**

$$
f_1(0) = \mathbb{E}[Y \mid X_1=0] - f_0 = \frac{0 + 1}{2} - 0.5 = 0
$$

$$
f_1(1) = \mathbb{E}[Y \mid X_1=1] - f_0 = \frac{1 + 0}{2} - 0.5 = 0
$$

\(f_2\)도 대칭적으로 \(f_2(0) = 0,\; f_2(1) = 0\).

XOR에서는 \(X_1=0\)의 평균 출력과 \(X_1=1\)의 평균 출력이 동일하다 (둘 다 0.5). 따라서 main effect는 정확히 0이다. 이것은 **XOR이 순수한 교호작용 함수**라는 직관과 일치한다 — 변수 하나만으로는 출력을 예측할 수 없으니까.

**2차 교호작용 — 하위 효과를 모두 빼고 남은 것:**

$$
f_{12}(x_1, x_2) = Y(x_1, x_2) - f_0 - f_1(x_1) - f_2(x_2)
$$

| \(f_{12}\) | \(X_2=0\) | \(X_2=1\) |
|---|---|---|
| \(X_1=0\) | \(0 - 0.5 - 0 - 0 = \mathbf{-0.5}\) | \(1 - 0.5 - 0 - 0 = \mathbf{+0.5}\) |
| \(X_1=1\) | \(1 - 0.5 - 0 - 0 = \mathbf{+0.5}\) | \(0 - 0.5 - 0 - 0 = \mathbf{-0.5}\) |

### fANOVA 분해 vs 1절의 네 가지 분해

fANOVA가 구한 분해를 1절의 네 분해와 비교하자:

| 분해 | \(f_0\) | \(f_1\) | \(f_{12}\) | fANOVA 제약 충족? |
|---|---|---|---|---|
| (a) AND형 | 0.25 | ±0.25 | [0.25, 0.75; 0.75, -0.75] | ✗ |
| (d) 순수형 | 0 | 0 | [0, 1; 1, 0] | ✗ |
| **fANOVA** | **0.5** | **0** | **[-0.5, 0.5; 0.5, -0.5]** | **✓** |

**1절의 네 가지 분해는 어느 것도 fANOVA 제약을 만족하지 않는다.** fANOVA 분해는 다섯 번째 — 유일한 — 분해다. 이 분해만이 아래에서 설명할 세 가지 제약 조건을 동시에 만족한다.

분해 (d)와 fANOVA는 main effect가 둘 다 0이라는 점에서 비슷해 보이지만, intercept가 다르다. (d)에서 \(f_0 = 0\)이고 \(f_{12}\)의 평균이 0.5인 반면, fANOVA에서는 그 0.5가 \(f_0\)로 옮겨져서 \(f_{12}\)의 평균이 정확히 0이 된다.

## 유일성을 보장하는 제약 조건

다음 세 가지 조건이 유일성을 보장한다.

### Zero Means (평균 0)

$$
\mathbb{E}[f_S(\mathbf{X}_S)] = 0 \quad \forall\, S \neq \emptyset
\tag{2}
$$

모든 component의 기대값이 0이다. 전체 평균은 \(f_0\) 하나에만 담긴다.

XOR 예시로 검증:

| component | 기대값 | 충족? |
|---|---|---|
| \(f_1\) | \((0 + 0) / 2 = 0\) | ✓ |
| \(f_2\) | \((0 + 0) / 2 = 0\) | ✓ |
| \(f_{12}\) | \((-0.5 + 0.5 + 0.5 + (-0.5)) / 4 = 0\) | ✓ |

반면 1절의 분해 (d)에서 \(f_{12}\)의 기대값 = \((0 + 1 + 1 + 0)/4 = 0.5 \neq 0\). Zero Means를 위반한다.

### Orthogonality (직교성)

$$
\mathbb{E}[f_S(\mathbf{X}_S) \cdot f_V(\mathbf{X}_V)] = 0 \quad \forall\, S \neq V
\tag{3}
$$

서로 다른 component는 직교한다. 정보가 겹치지 않는다.

XOR 예시에서 \(f_1 = 0\)이므로 \(f_1 \cdot f_{12} = 0\)은 자명하다. 일반적으로는 non-trivial한 조건이며, main effect가 0이 아닌 함수에서 중요해진다.

### Variance Decomposition (분산 분해)

$$
\sigma^2(\hat{f}) = \sum_{S \subseteq \{1,\ldots,p\}} \sigma^2(f_S)
\tag{4}
$$

총 분산 = 각 component 분산의 합. 직교성의 직접적 결과이다.

XOR 예시:

- 총 분산: \(\sigma^2(Y) = \mathbb{E}[(Y - 0.5)^2] = (0.25 + 0.25 + 0.25 + 0.25)/4 = 0.25\)
- \(\sigma^2(f_1) = 0\), \(\sigma^2(f_2) = 0\)
- \(\sigma^2(f_{12}) = (0.25 + 0.25 + 0.25 + 0.25)/4 = 0.25\)
- 합계: \(0 + 0 + 0.25 = 0.25\) ✓

이것은 **XOR의 모든 분산이 교호작용에서 온다**는 것을 의미한다. 변수 하나로는 XOR을 전혀 설명할 수 없으니 직관적으로도 맞다.

## 핵심: Integrate-to-Zero

위 세 조건 중 가장 실용적이고 강력한 형태:

$$
\forall\, u \subseteq [d], \quad \forall\, i \in u: \quad \mathbb{E}[f_u(X_u) \mid X_{u \setminus i}] = 0
\tag{5}
$$

**"교호작용 텐서의 모든 1차원 slice를 따라 가중평균을 내면 0이다."**

이것이 실질적으로 검사하는 것: **\(f_{12}\)에 \(f_1\)이나 \(f_2\)에 담겨야 할 효과가 남아 있지 않은가?**

### XOR 예시로 검증

fANOVA의 \(f_{12} = [-0.5, 0.5;\; 0.5, -0.5]\):

- \(X_1\)을 고정하고 \(X_2\)에 대해 평균:
    - \(X_1=0\): \((-0.5 + 0.5)/2 = 0\) ✓
    - \(X_1=1\): \((0.5 + (-0.5))/2 = 0\) ✓
- \(X_2\)를 고정하고 \(X_1\)에 대해 평균:
    - \(X_2=0\): \((-0.5 + 0.5)/2 = 0\) ✓
    - \(X_2=1\): \((0.5 + (-0.5))/2 = 0\) ✓

**모든 slice 평균이 0이다.** \(f_{12}\)에는 \(X_1\)만으로 설명 가능한 패턴도, \(X_2\)만으로 설명 가능한 패턴도 남아 있지 않다. 이것이 **순수한 교호작용**(pure interaction) — 두 변수가 **동시에** 특정 조합일 때만 나타나는, 더 적은 수의 변수로는 설명할 수 없는 효과다.

### 1절의 분해 (d)가 왜 실패하는가

비교: 분해 (d)의 \(f_{12} = [0, 1;\; 1, 0]\):

- \(X_2=0\) 고정, \(X_1\) 방향 평균: \((0 + 1)/2 = 0.5 \neq 0\) ✗

이 0.5는 \(X_2=0\)일 때의 평균 출력 — 즉 **\(X_2\)의 main effect 또는 intercept**에 해당하는 값이다. 이것이 \(f_{12}\)에 남아 있다는 것은 교호작용 텐서에 main effect가 **누출**되어 있다는 뜻이다.

fANOVA는 이런 누출을 허용하지 않는다. **교호작용에는 교호작용만, main effect에는 main effect만** — 이 원칙을 수학적으로 강제하는 것이 Integrate-to-Zero 조건이다.

---

이 세 조건을 동시에 만족하는 분해는 **유일(unique)**하다. 비로소 "이 변수의 효과가 양이다/음이다"를 신뢰할 수 있게 된다.

그런데 XGBoost나 EBM 같은 트리 기반 모형은 학습 과정에서 이 제약을 고려하지 않는다. 학습된 shape function이 이미 뒤섞여 있을 때, 이를 사후적으로 바로잡는 알고리즘이 필요하다. → [3절. Purification — 트리 모형의 사후 정제](purification.md)
