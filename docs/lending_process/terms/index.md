# 용어 정리

여신 프로세스 전반에서 반복적으로 등장하는 핵심 개념을 정리한다.
신용원가(Expected Loss)의 세 축인 **PD · LGD · EAD**를 각각 별도 페이지에서 다룬다.

$$
\text{Expected Loss} = \text{PD} \times \text{LGD} \times \text{EAD}
$$

| 약어 | 이름 | 의미 | 상세 |
|:---:|---|---|:---:|
| **PD** | Probability of Default | 차주가 부도날 확률 | [바로가기](pd.md) |
| **LGD** | Loss Given Default | 부도 시 손실률 | [바로가기](lgd.md) |
| **EAD** | Exposure at Default | 부도 시 익스포저 | [바로가기](ead.md) |
